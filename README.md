# NLP-модель для аналізу анотацій наукових публікацій

Даний проект є кваліфікаційною роботою бакалавра. Система призначена для автоматизованого семантичного аналізу наукових праць з репозиторію **ArXiv.org**.

## Основні функції
- **Асинхронний пошук:** отримання метаданих статей у реальному часі через ArXiv API.
- **Дворівневий NLP-конвеєр:** 
  1. Статистичний аналіз уніграм (бібліотека **spaCy**).
  2. Екстракція концептуальних підтем за допомогою трансформерної моделі **KeyBERT**.
- **Інтерактивна візуалізація:** побудова графіків частотності та семантичного ландшафту за допомогою **Recharts**.
- **Генерація звітів:** експорт результатів аналізу у формат PDF на стороні клієнта.

## Технологічний стек
- **Backend:** Python 3.10+, FastAPI, spaCy (en_core_web_lg), KeyBERT.
- **Frontend:** React, Axios, Recharts, jsPDF, html2canvas.

## Встановлення

### Backend (FastAPI)
Спочатку треба зайти в папку бекенду - `cd backend`
    Перед запуском проєкту встановіть необхідні бібліотеки:
        - `pip install pip install arxiv spacy keybert fastapi uvicorn`
    Також потрібно завантажити мовну модель spaCy:
        - `python -m spacy download en_core_web_sm`

### Frontend (React)
Спочатку треба зайти в папку фронтенду - `cd frontend/nlp-app`
    Встановіть основні залежності проєкту:
        - `npm install`
    Якщо необхідно встановити додаткові бібліотеки вручну:
        - `npm install axios jspdf html2canvas recharts`

## Запуск

### Backend (FastAPI)
Спочатку треба зайти в папку бекенду - `cd backend`
    - `python -m uvicorn nlp_app:app`

### Frontend (React)
Спочатку треба зайти в папку фронтенду - `cd frontend/nlp-app`
    - `npm start`
